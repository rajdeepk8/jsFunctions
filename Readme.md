# JavaScript - Arrays and Functions

created JavaScript functions that receive an array as argument and perform various operations on it, like finding the maximum number, calculating the sum of all elements, and counting the number of odd nnumbers.




## findMaximumNumber():
An Named Function is used to find the maximum number of an array.it takes an array as an input where using 'for' loop and 'if' condition the function provides an output which is maximum number in the array.

## sumOfElements():
This function is an Arrow Function which results sum of all elements in an array as output provided an array as input.

a variable named 'sum' is maintained and initialized with 0. Using a 'for' loop Each element of the array is added to the 'sum' variable one by one. And sum of all the element is calculated.

## countOdd():
An Anonymous Function that results count of odd numbers present in the array

One 'count' varibale is maintained with 0 as intialized value. Using 'for' loop each element in the array is checked whether it is divisible by 2 or not, if no then count is increased by 1
